# IoT Frontend Dashboard

Modern React-based monitoring dashboard for operating theater rooms with real-time sensor data.

## Quick Start

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```

## Default Login
- Admin: `admin` / `admin123`
- User: `user` / `user123`

## Tech Stack
React 18, TypeScript, Vite, TanStack Router, TanStack Query, Zustand, Shadcn UI, Tailwind CSS

See full README for complete documentation.
